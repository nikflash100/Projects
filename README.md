A compilation of my coding projects in Python and Java.

## From Failure to Flourishing
Data analysis of student performance using `student_data.csv`. Explores how absences, study time, parental support, and other factors affect academic failure. Built with pandas, matplotlib, and seaborn.

## Book Reader (Python)
Text analysis notebook (`Final_Project.ipynb`) that processes a book and computes word counts, palindromes, anagrams, frequency distributions, and more.

## Book Reader (Java)
Object-oriented Java version (`finalProjectOO.java`) of the book analysis project, with 16 text analysis methods.
